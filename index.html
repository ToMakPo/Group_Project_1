Rules

The goal of this game is to answer as many questions as possible at the hardest level to get the highest score and rule the Leader Board.

Game Setup: Player can pick a Category, Number of questions (from 4 up to 20), and select the Difficulty Level.

Earning Score:

  • Easy difficulty: Correct = +2 points, Wrong = -1 point
  • Medium difficulty: Correct = +4 points, Wrong = -2 points
  • Hard difficulty: Correct = +8 points, Wrong = -4 points

TIPS to earn as many point as possible:

  • Set the number of questions to 20 to earn as many points as you can.
  • Select the hard difficulty to earn the most points.
  • Read the questions carefully and get as many correct answers as possible.
  • That's it!

When the game is over, enter your initials and see where you place on the Leaderboard.

Play again to become a MASTER of Trick Tivia and own the leaderboard.
